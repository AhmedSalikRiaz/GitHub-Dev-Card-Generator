# GitHub Card Generator

> **Note:** There is a live demo currently available, but it is hosted on a personal cloud account and may be taken down in the future to conserve resources. If the live link is down, you can easily run this project locally on your own machine!
> 
> 🔗 **Live Frontend UI:** [https://github-card-frontend-482737836628.us-central1.run.app](https://github-card-frontend-482737836628.us-central1.run.app)  
> 🔗 **Live Backend API:** [https://github-card-backend-482737836628.us-central1.run.app](https://github-card-backend-482737836628.us-central1.run.app)

A full-stack application that generates personalized GitHub stat cards using AI. It consists of a FastAPI backend and a static HTML/JS frontend, fully containerized and ready for deployment on Google Cloud Run.

## 🚀 Features

- **AI-Powered Card Generation:** Uses Gemini AI to process and generate personalized GitHub profiles.
- **FastAPI Backend:** A lightweight and performant Python API.
- **Static Frontend:** A clean, responsive user interface.
- **Dockerized:** Includes `Dockerfile`s for both frontend and backend, along with a `docker-compose.yml` for easy local development.
- **Cloud Run Ready:** Configured for seamless deployment to Google Cloud Run.

## 🛠️ Local Development

### Prerequisites
- [Docker Desktop](https://www.docker.com/products/docker-desktop) installed.
- A Gemini API Key (`GOOGLE_API_KEY`)
- A GitHub Personal Access Token (`GITHUB_TOKEN`)

### Setup

1. **Configure Environment Variables:**
   Create a `.env` file in the `backend/` directory by copying `.env.example`:
   ```bash
   cp .env.example backend/.env
   ```
   Open `backend/.env` and fill in your keys:
   ```env
   GOOGLE_API_KEY=your_gemini_api_key_here
   GITHUB_TOKEN=your_github_token_here
   ```

2. **Run with Docker Compose:**
   Start the application locally using Docker Compose:
   ```bash
   docker-compose up --build
   ```

3. **Access the App:**
   - **Frontend:** Open your browser to `http://localhost:80`
   - **Backend API:** `http://localhost:8080`

## ☁️ Deployment (Google Cloud Run)

This project can be deployed as two separate scalable services on Google Cloud Run.

1. **Deploy the Backend:**
   ```bash
   gcloud run deploy github-card-backend --source ./backend --region us-central1 --allow-unauthenticated --set-env-vars "GOOGLE_API_KEY=...,GITHUB_TOKEN=..."
   ```

2. **Deploy the Frontend:**
   Take the URL generated by the backend deployment and pass it as the `BACKEND_URL`:
   ```bash
   gcloud run deploy github-card-frontend --source ./frontend --region us-central1 --allow-unauthenticated --set-env-vars "BACKEND_URL=YOUR_BACKEND_URL" --port=80
   ```
